Langkah mudah untuk menjalankan code di komputer Anda:

1. Pastikan Anda sudah menginstal Python di komputer.
2. Buka Terminal (Mac/Linux) atau Command Prompt / CMD (Windows).
3. Install library yang dibutuhkan dengan mengetik perintah ini:
   
   pip install scikit-learn pandas

4. Buka folder ini di text editor pilihan Anda (seperti VS Code).
5. Jalankan file .py mana saja langsung untuk melihat hasilnya!
